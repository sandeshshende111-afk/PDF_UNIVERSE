<script>
// Cookie consent logic
function setCookieChoice(accepted) {
  localStorage.setItem('cookie_consent', accepted ? 'accepted' : 'declined');
  localStorage.setItem('cookie_consent_date', new Date().toISOString());
  document.getElementById('cookieBanner').style.display = 'none';
  if (accepted) initAnalytics(); // load GA only if accepted
}

function checkCookieConsent() {
  const consent = localStorage.getItem('cookie_consent');
  const date    = localStorage.getItem('cookie_consent_date');
  // Show banner if no choice made, or choice was over 12 months ago
  const expired = date && (Date.now() - new Date(date).getTime()) > 365*24*60*60*1000;
  if (!consent || expired) {
    setTimeout(() => {
      document.getElementById('cookieBanner').style.display = 'flex';
    }, 1500); // show after 1.5s
  } else if (consent === 'accepted') {
    initAnalytics();
  }
}

// Call on page load
document.addEventListener('DOMContentLoaded', checkCookieConsent);
</script>

Google Analytics 4 Setup

This tells you how many people visit your site, which tools they use most, and where they come from. Essential for knowing if your marketing is working.

💡
Get your GA4 Measurement ID first:
1. Go to analytics.google.com
2. Click "Start measuring" → enter your site name
3. Choose "Web" → enter your Netlify URL
4. Copy the Measurement ID (looks like: G-XXXXXXXXXX)
5. Replace G-XXXXXXXXXX in the code below with your real ID
Paste inside <head> tag in pdfuniverse.html 
<!-- Google Analytics 4 — loads only after cookie consent -->
<script>
function initAnalytics() {
  // Replace G-XXXXXXXXXX with your real GA4 Measurement ID
  var GA_ID = 'G-XXXXXXXXXX';
  if (window.gaInitialized || !GA_ID || GA_ID === 'G-XXXXXXXXXX') return;
  window.gaInitialized = true;

  // Load GA4 script dynamically
  var script = document.createElement('script');
  script.async = true;
  script.src = 'https://www.googletagmanager.com/gtag/js?id=' + GA_ID;
  document.head.appendChild(script);

  window.dataLayer = window.dataLayer || [];
  function gtag(){ dataLayer.push(arguments); }
  window.gtag = gtag;
  gtag('js', new Date());
  gtag('config', GA_ID, {
    anonymize_ip: true,        // GDPR compliance
    cookie_expires: 180,       // 6 months instead of 2 years
    send_page_view: true
  });
}

// Track tool usage events (add to processTool() function)
function trackToolUsage(toolName) {
  if (typeof gtag !== 'undefined') {
    gtag('event', 'tool_used', {
      event_category: 'PDF Tools',
      event_label: toolName,
      value: 1
    });
  }
}

// Track Pro upgrade clicks
function trackUpgradeClick(source) {
  if (typeof gtag !== 'undefined') {
    gtag('event', 'upgrade_click', {
      event_category: 'Conversion',
      event_label: source
    });
  }
}
</script>
ℹ️
The initAnalytics() function is called by the cookie consent banner above — analytics only loads after the user clicks "Accept." This makes your site GDPR compliant by default.

Privacy-Friendly Alternative: Simple Analytics

If you don't want to deal with cookie consent at all, use Simple Analytics — it tracks visits without cookies or personal data, so no GDPR banner needed. Free for small sites.

No cookie consent needed — paste anywhere in <head> 
<!-- Simple Analytics — cookie-free, GDPR compliant by default -->
<!-- Sign up free at simpleanalytics.com, replace yourdomain.app with your domain -->
<script async defer src="https://scripts.simpleanalyticscdn.com/latest.js"></script>
<noscript><img src="https://queue.simpleanalyticscdn.com/noscript.gif"
  alt="" referrerpolicy="no-referrer-when-downgrade"/></noscript>
💡
Which should you choose?
• Google Analytics → More detailed data (traffic sources, demographics, conversion funnels). Requires cookie consent banner. Free.
• Simple Analytics → Basic page views and referrers. No cookies, no consent needed. $9/month but free for small sites.

How to Verify Everything Is Working

Test the Cookie Banner

1. Open your site in an incognito/private browser window
2. After 1.5 seconds, the cookie banner should appear at the bottom
3. Click "Accept" — the banner disappears and GA loads
4. Click "Decline" — banner disappears, GA does NOT load

Test Google Analytics

1. Accept cookies on your site
2. Go to Google Analytics → Reports → Realtime
3. You should see "1 user in last 30 minutes" within 60 seconds
4. If nothing shows, double-check your G-XXXXXXXXXX ID

What to Monitor in GA4

Reports → Acquisition → Traffic Acquisition → where do your visitors come from?
Reports → Engagement → Events → which tools are being used?
Reports → Monetization → Overview → upgrade clicks and conversions
Realtime → who's on your site RIGHT NOW
